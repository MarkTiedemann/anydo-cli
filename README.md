# anydo-cli

**Unofficial CLI app for [Any.do](https://www.any.do/).**

## Installation

```
npm install anydo --global
```

## Quickstart

```
$ anydo --help

  Any.do CLI app.

  - Login
    $ anydo login --email you@example.org --password super-secret

  - List your tasks
    $ anydo [tasks]

  - Logout
    $ anydo logout

```
